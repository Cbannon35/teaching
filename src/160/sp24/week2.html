querySelectorAll returns a bunch of elements, using any CSS selector

                for (const element of redElements) {
                    element.style.color = 'red';
                }

                const paragraphs = document.querySelectorAll('p');

                for (const element of paragraphs) {
                    element.textContent = 'I am a paragraph!';
                }

some APIs return a Promise, which can be given callbacks in a .then() call::


                fetch(
                  'https://poetrydb.org/...'
                ).then((data) => {
                  return data.json(); // this *also* returns a Promise
                }).then((json) => { // Promises can be chained!
                  console.log(json); // now we actually have the data
                });
https://codepen.io/Timothy-James-Aveni/pen/eYXVEvW

you can use the await keyword to suspend execution until a promise finishes.


                // Mark func *async* to use the `await` keyword.
                const url = '...';
                const getPoem = async () => {
                  const data = await fetch(url);
                  const json = await data.json();

                  console.log(json);

                  return json; // returns a Promise because it's async!
                };
                const poem = getPoem();
                console.log(poem); // not ready yet!
