What is React Native?

  • Developed by facebook
  • Allows you to build native mobile apps using React and Javascript

Benefits? Many!

  • Cross Platform Apps
  • Native performance: Renders native UI elements
  • Community Support

Isn't it just React?

Well yes.. but No!! When using React Native, you should use mobile platform specific components from the React Native library. Additionally, there are APIs that allow you to interact with the device hardware (camera, mic, etc).

More on Screens

  • Think of a stack:
    • Each screen is a layer on the stack
    • When you navigate to a new screen, it is added to the stack
    • When you go back, the top screen is removed from the stack
  • Check out NavigationContainer, reateNativeStackNavigator, createBottomTabNavigator
